Fetch API Functions

The following functions GET data from JSON Placeholder API.

Fetch Data by ID

Input which data set you'd like to retrieve and the ID of the data.
Choose from posts, comments, albums, photos and todos.

Sort Data by ID

Click the button to sort the data you have already fetched by its IDs.

Fetch Posts by User

Specify the user whose posts you'd like to fetch.
